---
title: Icon
description: Semantic SVG icon wrapper with multiple sizes.
---

### Usage

The `IconComponent` renders SVG assets from the `assets/icons` directory.

```html
<span icon name="bell" [size]="24"></span>
```

### API

- **`name`**: The filename of the SVG in `assets/icons` (without extension).
- **`size`**: The width/height in pixels (default: 24).
